# 🎓 Student Decision Helper

## 📖 Overview

**Student Decision Helper** is a web-based career guidance application designed to help students make informed academic and career decisions. The application analyzes user information such as age, current education, subject interests, and career preferences to provide personalized career recommendations.

The project offers a clean and user-friendly interface with a modern dark theme, making career exploration simple and interactive.

## 🚀 How to Run

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Run `login.html` using Live Server or any web browser.
4. Start the Core Java backend.
5. Register/Login and explore the application.
